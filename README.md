## 빌트인 캠을 활용한 소통형 AI 커넥티드 카 플랫폼
- 운전자가 파악한 차량의 내·외부 상황을 소통형 AI를 활용하여 분석

- 개인이 파악하지 못한 위험 상황을 커넥티드 카 플랫폼을 통하여 인지
